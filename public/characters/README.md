# Ilustraciones de personajes

Coloca aquí los PNG (fondo transparente o blanco) con estos nombres exactos.
Aparecerán automáticamente en la introducción y la historia; si falta alguno,
se muestra un avatar con emoji como respaldo.

| Archivo             | Personaje              | Descripción de tu ilustración             |
|---------------------|------------------------|-------------------------------------------|
| `funcionario.png`   | Don Ramiro (FIFA)      | Hombre calvo agobiado con torre de carpetas |
| `presidente.png`    | Presidente federación  | Traje azul marino señalando con el dedo   |
| `seleccionador.png` | Tú (seleccionador)     | DT con americana, pizarra y portátil      |
| `analista.png`      | Leo (analista)         | Joven con americana gris y tablet         |
| `veterano.png`      | El Míster              | Veterano en chándal con pizarra táctica   |

Tamaño recomendado: ~600×600 px (se recortan cuadradas). No subir imágenes con
derechos de terceros.
